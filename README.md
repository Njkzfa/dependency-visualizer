# Dependency Visualizer -  Инструмент визуализации графа зависимостей

## 📖 Общее описание

Инструмент для анализа и визуализации графа зависимостей пакетов Python. Программа позволяет:
- Анализировать зависимости пакетов из PyPI
- Строить граф зависимостей с учетом транзитивности
- Визуализировать результаты в различных форматах
- Работать в тестовом и производственном режимах
