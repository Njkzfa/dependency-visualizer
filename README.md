# Инструмент визуализации графа зависимостей 21 вариант

## Общее описание
Данный инструмент разработан для анализа и визуализации графа зависимостей пакетов в выбранной экосистеме. Он позволяет получать информацию о зависимостях, строить граф и выполнять с ним различные операции, включая визуализацию в нескольких форматах.

## Функциональность и настройки

### Этап 1. Минимальный прототип с конфигурацией
- Реализовано CLI-приложение с поддержкой конфигурационного файла в формате JSON.
- Настраиваемые параметры:
  - Имя анализируемого пакета
  - URL-адрес репозитория или путь к тестовому файлу
  - Режим работы с тестовым репозиторием
  - Версия пакета
  - Максимальная глубина анализа зависимостей
- Реализован вывод всех параметров в формате ключ-значение при запуске.
- Добавлена обработка ошибок для всех параметров.

### Этап 2. Сбор данных
- Реализована логика получения данных о зависимостях для формата пакетов Rust (Cargo).
- Информация извлекается для заданной версии пакета.
- Получены прямые зависимости пакета с использованием URL-адреса репозитория.
- Реализован вывод всех прямых зависимостей на экран.

### Этап 3. Основные операции
- Построение графа зависимостей с учетом транзитивности с использованием алгоритма BFS без рекурсии.
- Учтена максимальная глубина анализа, заданная пользователем.
- Обработаны случаи циклических зависимостей.
- Реализован режим тестирования с возможностью указания пути к файлу описания графа репозитория.

### Этап 4. Дополнительные операции
- Реализован режим вывода обратных зависимостей для заданного пакета.
- Использован алгоритм обхода из предыдущего этапа.
- Продемонстрирована функциональность на различных тестовых случаях.

### Этап 5. Визуализация
- Сформировано текстовое представление графа на языке диаграмм PlantUML.
- Реализован вывод изображения графа на экран.
- Продемонстрированы примеры визуализации для трех различных пакетов.
- Проведено сравнение с выводом штатных инструментов визуализации.
